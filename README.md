## API Reference

#### Get all users  

```http
  GET /api/users
```  


#### Add user

~~~http
  POST /api/user
~~~

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `usuario`  | `string` | **Required**. Name of a new item |
| `senha` | `string` | **Required**. Name of a new item |
